# Creative Incubator – Assessment 2

## Personal Portfolio Website
This project is a personal developer portfolio website created as part of Creative Incubator Assessment 2. The website showcases my skills, projects, and contact information using a modern and interactive design.

## Project Overview
The goal of this project is to build a responsive and visually engaging portfolio website that represents my work as a developer. The site includes animated effects, interactive UI elements, and structured sections to present information clearly.

## Features
* Animated gradient background
* Custom glowing cursor and cursor trail
* Floating profile image animation
* Glassmorphism styled sections
* Responsive navigation bar with mobile menu
* Smooth scrolling navigation
* Project cards with 3D hover effects
* Portfolio project filtering
* Project preview modal
* Scroll progress bar
* Back-to-top button
* Social media icon hover animations
* Auto-scrolling skills section
* Section reveal animations
* Theme toggle option
* Fully responsive design

## Website Sections
* **Hero Section** – Introduction and main call-to-action
* **About Section** – Information about me
* **Skills Section** – Technologies and tools I use
* **Projects Section** – Portfolio of my work
* **Contact Section** – Ways to get in touch
* **Footer** – Additional links and credits

## Technologies Used
* HTML5
* CSS3
* JavaScript
* Font Awesome Icons
* AOS (Animate On Scroll)

## Purpose of the Project
This project was developed for **Creative Incubator Assessment 2** to demonstrate my ability to:
* Design a responsive website
* Implement modern UI/UX features
* Use animations and interactive elements
* Organize and present content effectively

## How to Run the Project
1. Download or clone the project files
2. Open the project folder
3. Run **index.html** in a web browser
